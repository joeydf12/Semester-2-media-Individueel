Tijdens mijn persoonlijke project ben ik bezig geweest om in figma een website te maken van mijn eigen team hier heb ik dan ook veel verschillende versies gemaakt om uiteindelijk bij het definitieve ontwerp te komen.

    Ik heb geleerd door verschillende ontwerpen te maken je er uiteindelijk zelf en anders door feedback van andere erachter komt wat wel en wat niet past. Zo begon ik eerst met outro’s bij een tiktok filmpje. Na onderzoek, feedback en verschillende versies kwam ik erachter dat outro’s bij Tiktoks niet gebruikelijk is. Deze heb ik dan ook alleen maar bij Instagram reels en YouTube shorts gebruikt. Ook bij bijvoorbeeld mijn persoonlijke project begon ik met een saaie website na wat feedback kwam ik uiteindelijk bij een veel beter en modernere site.

    Development

    In sprint 1 ben ik elke woensdag bezig geweest met development hier hebben we verschillende opdrachten gemaakt. De grootste opdracht was om een website in elkaar te zetten met behulp van voornamelijk grid Ook ben ik bezig geweest met verschillende opdrachten om javascript te oefenen. Hier ben ik met de basis opdrachten begonnen omdat ik hier nog niet eerder mee heb gewerkt. Dit vind je terug in mijn git lab.

    Grid was de eerste keer dat ik dit gebruikten. Vooral als het gaat om uitlijning vond ik grid goed voor mij om mee te werken. Om dit beter onder de knie te krijgen moet ik nog meer met grid gaan oefenen. Ik merk nog heel erg dat ik javascript erg lastig vindt.     Tijdens het bouwen van project x vond ik dat ik veel vooruitgang maakte en merk dat het bouwen van sites me steeds beter afgaat. Ook het werken met Javascript lukt met de dag beter. Ik ben benieuwd hoe ik het mobiel responsive ga maken.

Als het gaat om het design in het project van de McDonald’s ben ik in de eerste sprint eigenlijk met twee designs bezig geweest dat van de kaarten die we gebruiken en dat van de penalty shootout. Bij het persoonlijke project heb ik ook een kaart van de meeste spelers gemaakt. Daarbij heb ik een website volledig gedesignd
